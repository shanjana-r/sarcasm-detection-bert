# Sarcasm Detection using BERT 🤖

## 📌 Overview

This project focuses on detecting sarcasm in text using BERT (Bidirectional Encoder Representations from Transformers), a powerful transformer-based NLP model. Sarcasm detection is challenging as it requires contextual understanding beyond literal meaning.

## 🎯 Objective

To classify whether a given sentence is sarcastic or not using deep learning techniques.

## 🧠 Model

* BERT (Hugging Face Transformers)
* Fine-tuned on labeled sarcasm dataset

## ⚙️ Tech Stack

* Python
* PyTorch
* Transformers
* Scikit-learn
* Pandas, NumPy

## 🔍 Methodology

* Data preprocessing and cleaning
* Tokenization using BERT tokenizer
* Model training and fine-tuning
* Evaluation using accuracy and F1-score

## 📊 Results

* Accuracy: 93%
* F1 Score: 0.92
